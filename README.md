# Budget Tracker (Python)

Petit projet Python en ligne de commande pour gérer ses dépenses.

## Fonctionnalités
- Ajouter une dépense
- Afficher les dépenses
- Calculer le total
- Sauvegarder dans un fichier texte

## Lancer le projet
```bash
python budget_tracker.py
```

## Compétences montrées
- Fonctions
- Conditions
- Boucles
- Gestion des erreurs
- Manipulation de fichiers
